# CBS Safety Intelligence

CBS Safety Intelligence is a lean safety operating system for manufacturers: a dashboard-first safety command center with mobile hazard reporting, AI hazard analysis, corrective actions, incident investigation, AI root cause tools, JHA builder, observations, training, QDIPS, recognition, analytics, and admin settings.

## Streamlit Cloud Deployment

Use this main file path in Streamlit Cloud:

```text
cbs-safety-intelligence/streamlit_app.py
```

Dependencies are in:

```text
cbs-safety-intelligence/requirements.txt
```

## Local Run

```bash
pip install -r cbs-safety-intelligence/requirements.txt
streamlit run cbs-safety-intelligence/streamlit_app.py
```

## Product Positioning

CBS Safety Intelligence - A Lean Safety Operating System for Manufacturers.

Built for small and mid-sized manufacturing companies that need fast, modern, practical safety software for plant managers, safety managers, supervisors, team leaders, and shop-floor employees.
