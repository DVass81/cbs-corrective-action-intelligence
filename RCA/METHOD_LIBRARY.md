# RCA Method Library

The app includes a broad practical set of root-cause and corrective-action methods commonly used across manufacturing, quality, safety, operations, and continuous improvement.

## Included Methods

- 5 Whys
- Fishbone / Ishikawa / Cause-and-effect diagram
- 5M+1E / 6M
- Pareto analysis
- FMEA
- Fault tree analysis
- 8D problem solving
- A3 problem solving
- DMAIC
- PDCA / PDSA
- Is / Is Not analysis
- Change analysis
- Barrier analysis
- Process mapping / SIPOC
- Timeline / event and causal factor chart
- Control chart / run chart
- Scatter / correlation analysis
- Check sheet / stratification
- Current reality tree
- Bowtie analysis

## Reference Basis

- ASQ lists RCA and quality tools including fishbone, Pareto, FMEA, 5 Whys, flowcharts, control charts, scatter diagrams, and related templates.
- Six Sigma/DMAIC practice commonly combines process mapping, Pareto, fishbone, 5 Whys, FMEA, statistical charts, and control plans.
- 8D problem solving commonly uses data collection, Pareto, fishbone, process maps, 5 Whys, and FMEA.
- Healthcare and safety RCA toolkits commonly use 5 Whys, event timelines, barrier analysis, cause-and-effect methods, and evidence-based closure checks.

The app should treat these tools as structured decision support, not automatic proof. A case should not close until the selected method output is backed by objective evidence.
