# RCA Project Folder

This folder keeps RCA-specific project material together.

The deployed Streamlit app is still at the repository root because Streamlit Cloud expects a clear main file path:

```text
streamlit_app.py
```

RCA-specific planning, notes, exports, and future templates can live here without breaking deployment.

## Current RCA Capabilities In The App

- Guided investigation interview.
- Investigation Workspace with RCA map and closure readiness score.
- AI-guided 5 Why draft.
- 5M+1E cause exploration.
- Lean countermeasure strategy recommendations.
- FMEA, Pareto, fault tree, process map starters.
- Generated corrective, preventive, containment, and verification actions.
- One-click creation of generated CAPA records.
- CAPA quality checker for weak action language.
- RCA method output included in executive reports.
- RCA method library covering common quality, Lean, Six Sigma, safety, and operations methods.
